Sustainably Dressed
Website for BoilerMake VII

Sustainably Dressed is a website that aggregates secondhand/thrift clothing listings from various websites into a single page which uses machine learning to more easily allow people to find sources of clothing that have a smaller environmental impact

  • Wrote a BeautifulSoup Python web scraper that automatically collects listings for second hand clothing
  • Automatically outputs aggregated data onto a single webpage for the user
  • Used Tensorflow to train an AI to be able to identify different types of clothing

Void Therapy
Mobile App for Hello World 2019

Void Therapy is a mobile game built on the concept of "screaming into the void, but the void screams back" -- a fun twist that allows you to vent frustration and relieve stress by "yelling into the void" and being comforted by the fact that you are not alone

  • Made in Android Studio written in Dart on Google's open-source UI SDK Flutter
  • Utilizes Firebase database to store and retrieve messages/rants
  • Features handcrafted animated sprites and levelling system to give the player a sense of progression and accomplishment
