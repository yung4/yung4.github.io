Purcle
CS307 Software Engineering I

Purcle, a portmanteau of 'Purdue Circle,' is a reddit-like forum for Purdue students to post about different topics built with a React frontend and Python Django backend hosted on Heroku

  • Allows user to post text or images to a certain topic, where they can follow and view different topic timelines. The user will be able to see posts from their followed topics in their main homepage timeline. Users are able to comment, like, and save posts. Users are also able to direct message, block, and view other users.
  • Uses PostgreSQL as database management for the Django backend
  • Uses Javascript and React with a Bootstrap-React framework for the frontend
  • Django backend serves an API for getting and storing all the data created by Purcle users
  • Frontend calls API using Axios to display data and pass it through different components
